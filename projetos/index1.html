Sistema Humadej


Este foi um sistema desenvolvido sob encomenda para a Igreja IEADERP da minha cidade. Contemplando todas as quatro operações do CRUD, foi implementado em Java, utilizando o banco de dados MySQL. O projeto foi concebido integralmente, desde a análise de requisitos até a construção e treinamento dos usuários.

O sistema foi inspirado no CRUD de uma agenda e foi fundamental para dar vida ao meu projeto de extensão da faculdade, que concluí com uma ótima nota no curso de Análise e Desenvolvimento de Sistemas. Esta experiência proporcionou-me uma significativa oportunidade de aprimoramento em diversas áreas do desenvolvimento e software.
